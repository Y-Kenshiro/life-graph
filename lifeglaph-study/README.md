# ディレクトリ構成
	- API: LLMのデータを取得するコード  
		- 必要に応じてAPIキーを各LLM名のファイルに書き込む  
	- Code: コード関連  
	    - 説明は各コード内に記載  
	- Evaluation_matrix: 評価指標を整理したもの  
	- Gemini: Geminiで生成したデータとその分析結果  
	- Prompt: 使用するプロンプト（人生体験のはサンプル）  
	- User: 参加者のデータ  
	- venv: pythonの仮想環境  
	- 卒研の時: 卒研のときのデータ(整理前のコード)  
	    - このフォルダにあるコードを実行するときは、実行ディレクトリをここに移すこと  
	    - ディレクトリ指定の関係で上記をしないとエラーが出ます  
	- 参考文献: 論文の参考文献（Scrapboxに載せてないやつ）

# ※人生体験の内容が含まれるファイルの扱い
	-  必ずpass付きzip変換すること
		- 深見が使用したソフト: 7zip（https://7-zip.opensource.jp/）
	- 削除するときは、ゴミ箱の中からも削除する

# venv環境の構築
	venvを有効化したうえで、以下を実行
	- pip install -r requirements.txt
	※新しくリリースされたモデルに変更する場合、各APIのライブラリをアップデートする必要がある
